# Veterinaria Frontend

Este proyecto es una **prueba de frontend** desarrollada con **HTML, CSS y JavaScript**, enfocada en una página web básica para una veterinaria.

El objetivo es mostrar la estructura y el diseño de un sitio sencillo.

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript (básico)

## Estructura del proyecto

El proyecto es intencionalmente simple y solo contiene dos vistas principales hasta el momento:

- `index.html` → Vista de página principal  
- `contact.html` → Vista de contacto  

No se utiliza backend ni base de datos. Toda la lógica se maneja del lado del cliente.

## Funcionalidades

- Página principal con navegación responsive.
- Menú adaptable para dispositivos móviles (checkbox + CSS).
- Enlaces a redes sociales.
- Vista de contacto con formulario.
- Validación básica de formulario usando JavaScript.
- Diseño responsive usando media queries.

> Nota:  
> No se guarda información en una base de datos ni se envían datos a un servidor.  
> El manejo de la información es únicamente demostrativo para aplicar uso de JS.

## Propósito del proyecto

- Prueba de conocimientos básicos de frontend.
- Maquetación con HTML y CSS.
- Uso inicial de JavaScript para interacción con el usuario.
- Estructuración simple de un proyecto web.
